    📱 ➜ Mobile

    Mobile, ou smartphones, são dispositivos portáteis que integram a funcionalidade de computação e telefone em um único sistema operacional, como Android ou iOS.

    ► Aplicativos Mobile

    • Authenticator Pro - Cliente de autenticação de dois fatores (2FA) para Android + Wear OS

    • AdAway - AdAway é um bloqueador de anúncios gratuito e de código aberto para Android.

    • Droid-Ify - Ótimo cliente F-Droid com Material UI.

    • Snapdrop Android - Cliente Android para compartilhamento local de arquivos via Snapdrop e Pairdrop

    • Web Video Cast - Web Video Caster permite transmitir conteúdo online e local na TV.

    • Lemuroid - Tudo em um emulador no Android! Com uma interface simples e prática.

    • Lawnchair - Lawnchair é um aplicativo gratuito e de código aberto para Android. ele porta os recursos do Pixel Launcher e apresenta opções avançadas de personalização.

    • QuickPic Gallery Mod - Aplicativo de galeria altamente customizável e com bastante funcionalidades.

    • ReadYou - Um leitor de RSS Android apresentado no estilo Material You.

    • Amaze File Manager - Gerenciador de arquivos de design de materiais para Android

    • Twire - Twire é um cliente Twitch alternativo e de código aberto para Android

    • SmartTube - Aplicativo sem anúncios para assistir vídeos do Youtube em TV Box Android

    • Xtra - Xtra é outro cliente Twitch alternativo e de código aberto para Android

    • Bromite - Bromite é um fork do Chromium com bloqueio de anúncios e aprimoramentos de privacidade.

    • Hail - Aplicativo que congela ou suspende aplicativos usando root ou shizuku.

    • URLChecker - O URLCheck atua como um intermediário incrivelmente personalizável e poderoso ao abrir links url.

    • Simple QR - Simple QR, um aplicativo simples e leve para digitalizar, criar e armazenar códigos QR.

    • Orbot - Tor no Android

    • xManager - App para instalar Spotify sem anúncios.

    • ReVanced - Um gerenciador que aplica patchs e compila o YouTube com vários Mods incluindo o mod que tira os anúncios. (Necessário um APK do Youtube)

    • Android Cache Cleaner - Aplicativo que permite limpar o cache de todos os aplicativos de uma vez.

    • Aliucord - Uma modificação para o aplicativo Discord do Android.

    • Lucky Patcher - Desbloqueia compras no aplicativo em aplicativos Android e remove anúncios.

    • nzb360 - Usenet e aplicativo de gerenciamento remoto de torrent com suporte para gerenciamento de Sonarr / Radarr / Lidarr / Bazarr / Sickbeard / Couchpotato / Fones de ouvido

    • iTorrent Search - Aplicativo de busca de torrents para Android

    • iDope Torrent Search - Aplicativo de busca de torrents para Android

    • Libgen para Android - aplicativo de pesquisa/download do Libgen

    • Librebook para Android - Aplicativo de pesquisa/download do Libgen
